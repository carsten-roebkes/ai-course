# AI Course — from Data Analyst to AI Automation Engineer

My hands-on journey learning to build AI-powered automations.
A new small project lands here roughly every week.

## Projects so far

- **first_call.py** — my first live call to an LLM API; prints the model's answer and token usage.
- **explain5.py** — an "explain-like-I'm-five" tool: type any topic, get a simple 3-sentence explanation back from the model.

## Built with

- Python 3.14
- Google Gemini API (`google-genai`)
